## üß¨ Models and Datasets

### 1. **K-Means Clustering**
- **Dataset:** Malaria Cell Images  
- **Description:** Clustering parasitized vs uninfected cells.  
- **Dataset Link:** [Malaria Dataset](https://drive.google.com/file/d/13mDeOCSZbmikVFkZqwIIfQJMmIunUr1W/view?usp=sharing)  
- **Files:** `K-Means Clustering model.ipynb`, `K-Means Streamlit.py`

### 2. **Logistic Regression**
- **Dataset:** Malaria Cell Images  
- **Description:** Classification of parasitized vs uninfected cells.  
- **Dataset Link:** [Malaria Dataset](https://drive.google.com/file/d/13mDeOCSZbmikVFkZqwIIfQJMmIunUr1W/view?usp=sharing)  
- **Files:** `Logistic Regression model.ipynb`

### 3. **K-Nearest Neighbors (KNN)**
- **Dataset:** Credit Dataset  
- **Description:** Predicting health insurance costs.  
- **Dataset Link:** [Credit Dataset](https://drive.google.com/file/d/1no10Tf_nnfzrtLMtQRIGQVJgDDWbbyS6/view?usp=sharing)  
- **Files:** `KNN model.ipynb`, `KNN Streamlit.py`

### 4. **Linear Regression**
- **Dataset:** Credit Dataset  
- **Description:** Predicting health insurance costs using linear regression.  
- **Dataset Link:** [Credit Dataset](https://drive.google.com/file/d/1no10Tf_nnfzrtLMtQRIGQVJgDDWbbyS6/view?usp=sharing)  
- **Files:** `Linear Regression model.ipynb`
